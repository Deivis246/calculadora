OPCIÓN 1 — Abrir directamente en el navegador (más fácil)
──────────────────────────────────────────────────────────
1. Abre la carpeta del ZIP
2. Haz doble clic en el archivo  index.html
3. Se abre en tu navegador — listo ✓

   ⚠ Algunos navegadores bloquean archivos locales.
     Si no carga bien, usa la Opción 2.


OPCIÓN 2 — Servidor local con Node.js
──────────────────────────────────────
Requisitos: Node.js instalado (https://nodejs.org)

1. Abre una terminal en esta carpeta
2. Ejecuta:
      node server.js

3. Abre en tu navegador:
      http://localhost:3000


OPCIÓN 3 — Sin instalación: usar VS Code Live Server
─────────────────────────────────────────────────────
1. Instala VS Code + extensión "Live Server"
2. Abre esta carpeta en VS Code
3. Clic derecho en index.html → "Open with Live Server"


NOTA IMPORTANTE — Google Sheets
────────────────────────────────
El botón "Guardar en Google Sheets" NO funcionará fuera
de la plataforma Replit, ya que requiere su infraestructura.
Todas las calculadoras médicas SÍ funcionan completamente
en modo local.

Para usar la integración con Google Sheets, accede a la
versión en línea en: https://replit.com/@TuUsuario/tu-repl
